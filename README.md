# Interactive Resume Webpage

This project is an interactive resume webpage designed to showcase personal information, skills, experience, and education in a visually appealing manner.

## Project Structure

```
interactive-resume
├── index.html        # Main HTML document for the interactive resume
├── styles            # Directory containing CSS styles
│   └── style.css     # Styles for the webpage
├── scripts           # Directory containing JavaScript files
│   └── script.js     # JavaScript for interactivity
└── README.md         # Documentation for the project
```

## Getting Started

To get started with the interactive resume webpage, follow these steps:

1. **Clone the Repository**
   Clone this repository to your local machine using the following command:
   ```
   git clone <repository-url>
   ```

2. **Open the Project**
   Navigate to the project directory:
   ```
   cd interactive-resume
   ```

3. **Open `index.html`**
   Open the `index.html` file in your web browser to view the interactive resume.

## Features

- **Personal Information**: Displays your name, contact details, and a brief introduction.
- **Skills Section**: Highlights your key skills with visual indicators.
- **Experience Section**: Lists your work experience with descriptions and dates.
- **Education Section**: Showcases your educational background.

## Customization

Feel free to customize the content in `index.html`, adjust styles in `styles/style.css`, and enhance interactivity in `scripts/script.js` to fit your personal branding.
